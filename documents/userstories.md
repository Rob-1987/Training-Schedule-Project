User Stories: 


As Jason, I want to be able to select a fitness level, so that he can start and progress his fitness journey in a safe and practical way. 


As Jason, I want to be able to input my name in the product, so that it feels more personalised.


 As Jason, I want to be able to see the available days for exercise, so I can get started with planning my schedule.


Tasks
1.	Create 4 buttons for: beginner, intermediate, hard, advanced at the top section of the screen
2.	Create an input box where the user can type in their name and save
3.	Create buttons with days of the week on them so you can then view daily schedules upon clicking them








USER STORIES


As Rachel, I want to be able to choose a fitness technique, so that she can exercise in the comfort of her home.


As Rachel, I want to be able to do chest, back, legs, arms and 2 days of cardio with one day rest each week, so that I know my schedule and can plan my week and make sure I don’t over train muscle groups.


As Rachel, I want to be able to click a muscle group and see the exercises I have to do on that day, so that I have a clear understanding of what I have to do and if I don’t know then I can google it and learn the exercise.
