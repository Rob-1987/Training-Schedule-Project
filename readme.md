# Fitness App
A simple fitness tracking application that helps users log their workouts and monitor their progress over time.
This web application is built for users who have a busy lifestyle and want to maintain their fitness routine this web app is all about simplicity and ease of use for those who cant afford complex fitness apps or gym memberships.
## Features
-input form for your name so that it feels like a personal experience
-you can choose from a two of workout types, body weight, strength training, and cardio/HIIT
-this web app gives you a simple workout routine based on your selected workout type
-it provides a clear guide for each exercise to help you stay on track
-it allows you to see how many workouts you've completed
-it allows you to see the sets and reps for so that you can track your progress
-it has a simple and clean user interface for easy navigation
## Technologies Used
